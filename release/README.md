# Ex3 – Pac-Man (Intro to Computer Science)

This folder contains a runnable release of the **Ex3 Pac-Man** project.

---

## How to Run

### Windows
Double-click:
run.bat

### Any OS (manual)
Make sure Java is installed, then run:
```bash
java -jar Ex3_PacMan_Lidor.jar
```

---

## Game Start

Choose a level (0–2) from the console.

Choose control mode (Manual / AI).

Press SPACE to start the game.

---

## Controls

WASD / Arrow Keys – Move Pac-Man (Manual mode)

SPACE – Start the game

T – Toggle AI / Manual mode during runtime

---

Enjoy!
